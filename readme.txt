Add links in menu from Web-Tools.pl Link managment system.

== Description ==

This plugin adds a custom links from Web-Tools.pl Link Management which are displayed on your sidebar

== Installation ==

1. Upload the `web-tools-pl-zaplecze` folder to the `/wp-content/plugins/` directory
2. Activate the Zaplecze Web-Tools.pl Widget plugin through the 'plugins' menu in WordPress
3. Configure the plugin by going to `Appearance > Widgets` in the WordPress Admin Dashboard, drag the Zaplecze Web-Tools.pl Widget to a Sidebar or Widget area, and start filling out the details.

= I installed and activated your plugin Widget, but how do I configure it? =

To configure the new Widget (Zaplecze Web-Tools.pl), go to `Appearance > Widgets` in the WordPress Admin Dashboard, drag the Widget to a Sidebar or Widget area on the right, and start filling out the details.
